# Clothing Inventory Management System

A console based clothing inventory management system implemented in C++.

## Features
- Add clothing items to inventory
- Display all items
- Undo last added item using stack
- Menu-driven interface

## Data Structures Used
- Singly Linked List
- Stack

## Technologies
- C++
- Data Structures
- Dynamic Memory Allocation

## How to Run
Compile and run using any C++ compiler.
